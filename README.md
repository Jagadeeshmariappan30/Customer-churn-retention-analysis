# Customer Churn & Retention Analysis

## Overview
This project analyzes customer churn in a retail e-commerce dataset using RFM (Recency, Frequency, Monetary) analysis and customer segmentation techniques.

## Tools Used
- Python (Pandas, Google Colab)
- SQL
- Power BI

## Key Features
- Customer churn identification
- RFM-based segmentation
- Revenue loss analysis
- Business insights and recommendations

## Files Included
- Dataset (CSV)
- Google Colab Notebook (.ipynb)
- SQL queries (.sql)
- Power BI dashboard (.pbix)
- Final Report (PDF)

## Outcome
Identified high-risk customers and provided actionable strategies to improve retention and reduce revenue loss.
